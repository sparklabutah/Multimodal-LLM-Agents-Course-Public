## Multimodal LLM Agents — A Self-Paced Course

This is a free, self-paced version of **CS 6960: Multimodal LLM Agents**, a graduate course taught by [Kenneth Marino](https://kennethmarino.com) at the [University of Utah](https://www.cs.utah.edu/). All lecture slides, readings, and exercises are freely available here.

---

## About This Course

This course explores the rapidly developing area of multimodal large language models deployed in embodied, agentic settings. You'll learn the foundations of reinforcement learning and large language models, and understand how large-scale models can be applied to real-world environments.

**Topics include:**
- RL and LLM / VLM foundations
- Agent frameworks (ReAct, chain-of-thought, reflection)
- Retrieval-augmented generation (RAG) and memory
- Tool use and function calling
- Coding agents
- Agent evaluation
- Assistant agents, game agents, computer use, and robotics
